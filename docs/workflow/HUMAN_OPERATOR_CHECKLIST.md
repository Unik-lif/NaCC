# Human Operator Checklist

Before each round, quickly check:

- Did I update `CURRENT_STATE.md`?
- Did I create or reuse a task packet for this bounded implementation round?
- Am I asking for planning, implementation, log analysis, or paper scouting?
- Did reviewer read the code before I spent time on first-pass diff review?
- Did I send the request to the right agent?
- Did I paste a large raw log into a non-log-analyzer session?
- Am I asking for implementation without providing a clear ticket?
- Is this round exploratory or a committed task?
- Are the plan or conclusions I am referencing already written into `CURRENT_STATE.md` / `NEXT_STEPS.md` / `HYPOTHESES.md`?
- Did architecture discussion leak into a coding session?
- Did the scope drift without being recorded?

If two or more answers are "no" or "not sure", update state first and then continue.
