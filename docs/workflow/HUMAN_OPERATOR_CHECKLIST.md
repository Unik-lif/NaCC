# Human Operator Checklist

每轮开始前，先快速过一遍：

- 我更新了 `CURRENT_STATE.md` 吗？
- 我这次是要规划、实现、看日志，还是筛论文？
- 我把请求发给了对的 agent 吗？
- 我是不是把大段原始日志贴进了非 log analyzer 会话？
- 我是不是要求实现，但没有给 clear ticket？
- 我这次讨论是 exploratory 还是 committed task？
- 我引用的计划或结论，已经写进 `CURRENT_STATE.md` / `NEXT_STEPS.md` / `HYPOTHESES.md` 了吗？
- 我是不是在 coding 会话里混进了架构讨论？
- 本轮 scope 有没有悄悄变化但还没记录？

如果有两项以上回答为“没有”或“不确定”，先补状态，再继续。
